MyBB Discord Server Rules

  • The MyBB Code of Conduct is to be observed at all times.
  • Official MyBB support is not provided within the server. The sole place where official support is provided is the MyBB Community Forums.
  • Follow all rules that apply to the MyBB Community Forums. You can remind yourself of these rules here.
  • Please only speak English within the server.
  • Do not spam or flood the server. Please use a paste bin (like Laravel.IO/bin) to post code.
  • Bug reports and vulnerabilities must not be reported in the server. Bug reports should be reported in the correct Bug Reports forum and vulnerabilities through Private Inquiries.
  • If you have an issue with someone in the server you may speak to one of the MyBB Team members. All MyBB Staff have a seperate username color.
